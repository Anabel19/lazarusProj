# MiniLab

## Descripción
Este proyecto contiene contiene funciones muy básicas, resuelve ecuaciones diferenciales ordinarias y grafica, encuentra integral-área bajo la curva, solución de ecuación no lineal, gráfica de funciones 2D.

## Lazarus

Entorno de desarrollo integrado visual multiplataforma [(IDE)](https://www.lazarus-ide.org/) que se utiliza en este programa.

## Comandos

- Al ejecutar el programa muestra al costado derecho una lista de comandos con su respectivo ejemplo para facilitar el test.

- Esos comandos se colocan en la terminal al ejecutar el programa. Tenemos como ejemplo:

```bash
MiniLab--> edo(''x*y'',2,4,4,0)
```

- Para interpolar y extrapolar se hace uso de .txt, .csv:

```bash
MiniLab--> extrapolar('recta3.csv')
```

- Para poder realizar la gráfica se necesita colocar el comando:

```bash
MiniLab--> plot
```
